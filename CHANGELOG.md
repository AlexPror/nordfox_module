# Changelog

## 0.1.4
- Комплектовщик: переименование с учётом перестановок имён в таблице; проверка на совпадение целевых имён; экспорт `.frw` с ограничением **38** строк данных на фрагмент таблицы.
- Штампы: сбор `.cdw` рекурсивно по подпапкам; пакетное обновление с кнопкой остановки и `processEvents` для UI; доработка отключения автонумерации листа в документе.
- GUI: опция **автокомплекта** после открытия папки проекта (загрузка, переименование, штампы, `Перечень_чертежей_комплекта.frw`).

## 0.1.3
- Комплектовщик: вкладки «Перечень чертежей» и «Автонумерация», экспорт перечня в `.frw`, правила наименования в перечне, узлы — строка из ячейки материала штампа (через КОМПАС).
- Штампы: отключение автонумерации листа в документе КОМПАС, поля листа для комплекта (`SheetFirstNumber` / `SheetsCount`), литера в автонумерации листов.
- PDF: объединение через `PdfWriter` (pypdf 6), сортировка чертежей по «(лист N)» в имени перед экспортом.
- Проект: открытие папки без `.a3d`; экспорт PDF/DWG по явным путям источника/вывода; подсказки в UI; скрипт диагностики COM; `table_frw/README.txt`; дорожная карта `docs/roadmap-1.0.md`.

## 0.1.2
- Синхронизация вхождений в сборке: изолированный процесс, более безопасное сопоставление, понятная статистика в логе.
- Переменные и штампы: нормализация разбора блоков и метаданных, сохранение уже открытых в КОМПАС документов, карта ячеек штампа для надёжных операций «копировать и обновить» и обновления штампов.
- Экспорт: отдельные вкладки **PDF** и **DWG** через локальный сервис, независимые папки источника и вывода.
- PDF: устойчивость экспорта/слияния, диагностика ошибок слияния в интерфейсе и README.
- DWG: засчёт успешного экспорта по факту создания файла при ложноотрицательных ответах COM `SaveAs`.
