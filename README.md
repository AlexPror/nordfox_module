# NordFox Module Manager

Desktop-приложение для работы с проектами NordFox в **КОМПАС-3D**:

- сканирование переменных сборки/деталей/чертежей и динамическая форма редактирования
- каскадное обновление переменных **(сборка → детали → финальная пересборка сборки)** для корректного пересчёта геометрии
- режим безопасного тестирования: **«копировать и обновить»**
- обновление штампов чертежей
- генерация QR-кодов (PNG)

## Версия

Текущая версия: `0.1.1`

## Требования

- Windows
- Установленный КОМПАС-3D (COM API)
- Python 3.x

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
py -3 main.py
```

или через `run.bat`.

## Использование (основной сценарий)

1. Выбери папку проекта в UI.
2. Измени значения переменных на вкладке «Переменные».
3. Для безопасного теста включи «Работать с копией проекта» и укажи папку назначения.
4. Нажми «Копировать и обновить».
5. В правой панели можно:
   - скопировать подробный лог (Ctrl+C внутри окна лога),
   - сохранить лог в файл,
   - открыть сохранённый лог.

## Почему каскадное обновление

В ряде проектов ассоциативные связи/гиперссылки в деталях или видах могут не пересчитываться автоматически.
Поэтому выполняется каскад:

1) обновление переменных в сборке,  
2) обновление переменных в деталях,  
3) финальная пересборка сборки.
